FRUGALEATER

https://frugaleater.herokuapp.com/

Frugaleater is an app for finding cheap eats, specifically food items that are $10 or less.

FEATURES

* Users can sign up
* Users can search by food item
* Users can favorite a restaurant and view all of their favorites.


TECHNOLOGIES: Ruby on Rails, HTML5, CSS, Javascript
GEMS: Devise, Typheous
APIS: Locu
